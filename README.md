# Pong Game

A basic pong game using SVGs. This was our third project in the Web Development Program at Red Academy. 

## Setup

**Install dependencies:**

`> npm i`

**Run locally with Webpack Dev Server:**

`> npm start`

**Build for production:**

`> npm run build`

## Keys

**Player 1 (left):**
* a: up
* z: down

**Player 2 (right):**
* ▲ : up
* ▼: down
